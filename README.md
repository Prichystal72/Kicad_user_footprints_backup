
# KiCad Symbol Library (English)

A set of footprints for personal use.

## Purpose

- Share and synchronize symbols between multiple computers using GitHub.
- Serve as a base for comparing and managing custom symbols.

## Structure

Each `.pretty` file contains one or more symbol libraries.

## Recommendations

- Always make a backup before making changes.
- Use meaningful names for symbols and libraries for clarity.
- If you add new symbols, update this file with a description of the changes.




# KiCad Symbol Library

Sada footprintů pro vlastní užití. 

## Účel

- Sdílení a synchronizace symbolů mezi více počítači pomocí GitHubu.
- Základ pro porovnání a správu vlastních symbolů.

## Struktura

Každý `.pretty` soubor obsahuje jednu nebo více knihoven symbolů.

## Doporučení

- Před úpravami vždy proveďte zálohu.
- Pro přehlednost používejte smysluplné názvy symbolů a knihoven.
- Pokud přidáváte nové symboly, aktualizujte tento soubor s popisem změn.
